# 🖼️ Vision-Based Social Grid Automator (AutoGrid)
> **Focus:** Computer Vision, Marketing Automation, & Image Processing

## 🚀 Business Purpose
Automates social media grid generation and branding for marketing teams, replacing hours of manual Photoshop work with a single execution script.

---

## 🇹🇭 จุดประสงค์ของโครงการ (Business Purpose)
ระบบจัดการและตกแต่งรูปภาพอัตโนมัติสำหรับการตลาดโซเชียลมีเดีย ช่วยลดภาระงานของทีมคอนเทนต์ในการจัดระเบียบรูปภาพ การใส่ลายน้ำ และการสร้าง Layout (Grids) สำหรับ Facebook/Instagram โดยใช้ระบบประมวลผลภาพอัจฉริยะ
