# ใบเสนอราคา ระบบฝากเหล้าดิจิทัล (Bottle Keep System)

**เรียน ผู้ประกอบการและผู้บริหารร้านอาหาร**

ทางเรานำเสนอระบบจัดการการรับฝากเครื่องดื่มและจองโต๊ะในรูปแบบเว็บแอปพลิเคชัน (Web Application) ที่เชื่อมต่อกับระบบ LINE Official Account ของทางร้านโดยตรง (White-label) เพื่อยกระดับการให้บริการ ป้องกันการทุจริต และช่วยกระตุ้นยอดขายอย่างมีประสิทธิภาพ

## 2. อัตราค่าบริการ (Pricing Structure)

เพื่อความครอบคลุมในการบริหารจัดการงบประมาณ ทางเราขอนำเสนออัตราค่าบริการ 2 รูปแบบ ได้แก่ **รูปแบบเช่าบริการรายปี** และ **รูปแบบซื้อขาด (Lifetime)**

### 2.1 รูปแบบเช่าบริการ (Subscription Model)
*เหมาะสำหรับร้านที่ต้องการเริ่มต้นใช้งานระบบโดยลดเงินก้อนแรกในการลงทุน*

*   **Tier 1: Starter Package** —— ราคา **800 บาท / เดือน** (ชำระล่วงหน้า 3 เดือน)
*   **Tier 2: Professional Package** —— ราคา **1,500 บาท / เดือน** (ชำระล่วงหน้า 3 เดือน)
*   **Tier 3: Enterprise Package** —— *(ไม่มีบริการให้เช่ารายเดือน เนื่องจากต้องตั้งค่าเซิร์ฟเวอร์ส่วนตัว)*

### 2.2 รูปแบบซื้อขาด (Lifetime License) - *แนะนำพิเศษ* 🔥
*จ่ายครั้งเดียวจบ คุ้มทุนตั้งแต่เดือนที่ 9! ไม่มีค่าภาระผูกพันรายเดือนกวนใจ*

*   **Tier 1: Starter Package** —— ราคา **3,900 บาท** (ชำระครั้งเดียว)
*   **Tier 2: Professional Package** —— ราคา **12,900 บาท** (ชำระครั้งเดียว - *คุ้มกว่าเช่าเพียง 8 เดือนครึ่ง!*)
*   **Tier 3: Enterprise Package** —— ราคา **35,000 บาท** (ชำระครั้งเดียว - *สิทธิ์จัดการเซิร์ฟเวอร์ 100%*)

## 3. วิเคราะห์ความคุ้มค่า (Cost Comparison: Rental vs Lifetime)

### ตารางวิเคราะห์ต้นทุนเฉลี่ยต่อวัน (Daily Cost Analysis)

## 4. ค่าบำรุงรักษาระบบ (Maintenance Agreement - MA)

สำหรับลูกค้าที่เลือกซื้อในรูปแบบ **ซื้อขาด (Lifetime)** ทางเรามีบริการสนับสนุนทางเทคนิคฟรี 1 ปีแรก หลังจากพ้นระยะเวลา 1 ปี ทางร้านสามารถเลือกต่อสัญญาค่าบำรุงรักษาระบบ (MA) ได้ เพื่อให้ระบบได้รับการอัปเดตและมีผู้เชี่ยวชาญดูแลตลอดเวลา

*   **ค่า MA รายปี (ปีที่ 2 เป็นต้นไป): 3,900 บาท / ปี** (เฉลี่ยเพียง 325 บาท/เดือน)

**บริการที่รวมอยู่ใน MA:**
1. บริการให้คำปรึกษาและแก้ไขปัญหาทางเทคนิค (Technical Support)
2. อัปเดตความปลอดภัยและการเชื่อมต่อระบบ (Security & API Updates) กับเซิร์ฟเวอร์ Cloudflare และ LINE
3. รับประกันระบบออนไลน์ (Uptime Guarantee) 99.9%

*(หมายเหตุ: หากทางร้านไม่ประสงค์ต่อสัญญา MA ระบบยังคงสามารถใช้งานได้ตามปกติ เพียงแต่จะไม่ได้รับการอัปเดตระบบปฏิบัติการส่วนกลาง หรือบริการช่วยเหลือฉุกเฉิน)*
