# คู่มือการใช้งานระบบ V88 3-Tier Management System

ระบบนี้ถูกออกแบบมาเพื่อช่วยให้คุณสามารถสร้าง (Generate) และจัดการระบบฝากเหล้าให้กับหลายๆ ร้านค้าได้ในที่เดียว โดยแบ่งระดับฟีเจอร์ออกเป็น 3 ระดับ (Tier) ตามความต้องการของแต่ละร้าน

---

## 📂 โครงสร้างไฟล์ในโฟลเดอร์ V88_3_tier
1.  **`generator.html`**: หน้าจัดการหลัก (Configurator) สำหรับเลือก Tier และใส่ข้อมูลร้านค้า
2.  **`master_app.html`**: ไฟล์ระบบหลักที่รองรับทั้ง 3 Tier (ใช้ไฟล์นี้เป็นต้นแบบในการ Deploy)
3.  **`profiles.json`**: ไฟล์ฐานข้อมูลสำหรับเก็บรายชื่อร้านค้าทั้งหมดที่คุณดูแล

---

## 🛡️ รายละเอียดของแต่ละระดับ (Tiers)
*   **Tier 1 (Basic Member)**: 
    *   เน้นความรวดเร็วและประหยัดพื้นที่
    *   ฝาก-เบิกเหล้าได้ปกติ แต่ **ไม่มีการถ่ายรูปภาพ** (ใช้ไอคอนแทน)
    *   หน้าจอเรียบง่าย (Minimalist)
*   **Tier 2 (Standard Photo)**: 
    *   สามารถ **ถ่ายรูปขวดเหล้า** จริงแนบลงไปในการฝากได้
    *   ช่วยป้องกันการสับสนของขวดที่ยี่ห้อซ้ำกัน
*   **Tier 3 (Premium CRM)**: 
    *   มีระบบ **สะสมแต้ม (Points)**
    *   มีระบบ **คูปองรางวัล (Rewards)**
    *   พนักงานสามารถจัดการแต้มและคูปองให้ลูกค้าได้ทันทีในหน้าค้นหา

---

## 🚀 ขั้นตอนการเริ่มสร้างร้านใหม่
1.  เปิดไฟล์ **`generator.html`** ด้วยเบราว์เซอร์
2.  **กรอกข้อมูลร้าน**: ใส่ชื่อร้าน, LIFF ID และ URL ของ Worker ที่คุณ Deploy ไว้
3.  **เลือก Tier**: กดเลือก 1, 2 หรือ 3 ตามแพ็กเกจที่ต้องการให้ร้านนั้นๆ
4.  **บันทึกข้อมูล**: 
    *   กดปุ่ม **"SAVE PROFILE"** เพื่อเก็บข้อมูลลงในรายการด้านล่าง (ข้อมูลจะบันทึกลงในเครื่องของคุณเพื่อให้เรียกใช้ง่ายในครั้งต่อไป)
5.  **นำไปใช้งานจริง**: 
    *   กดปุ่ม **"GENERATE CONFIG"**
    *   คัดลอกโค้ดที่ได้ ไปวางทับในส่วน `const CFG = { ... }` ในไฟล์ **`master_app.html`** (หรือไฟล์ที่คุณตั้งชื่อใหม่ให้ร้านนั้น)
