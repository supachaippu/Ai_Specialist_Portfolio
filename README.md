## 🚀 How to use this Portfolio (วิธีใช้งาน)
Each project folder is self-contained. To explore a project:
1. **Navigate** to the specific project directory (e.g., `cd 01-NLP-Driven-Agenda-Automation`).
2. **Read the internal `README.md`** for detailed technical specs and local setup instructions.
3. **Environment Setup:** Most projects require environment variables (API Keys) which are currently replaced with placeholders for security.

ทุกโฟลเดอร์โครงการเป็นแบบแยกส่วน (Self-contained):
1. **เข้าไปยังโฟลเดอร์** ของโครงการที่สนใจ (เช่น `cd 01-NLP-Driven-Agenda-Automation`)
2. **อ่านไฟล์ `README.md` ด้านใน** เพื่อดูรายละเอียดทางเทคนิคและวิธีการรันระบบ (Setup)
3. **การตั้งค่า:** โครงการส่วนใหญ่ต้องใช้ API Key ซึ่งปัจจุบันถูกแทนที่ด้วย Placeholder เพื่อความปลอดภัย กรุณาใส่ Key ของคุณเองก่อนใช้งาน
